# Truman Virtual Tour

An immersive 3D virtual tour application for Truman State University that allows prospective students and visitors to explore the campus through an interactive browser-based 3D environment.

## Revamped Architecture (2026)

This project has been rebuilt from "Ground Zero" with a modern stack to improve understandability, performance, and scalability.

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **3D Engine**: React Three Fiber (Three.js)
- **Deployment**: Vercel

---

## Overview

This application provides a complete virtual campus experience with:
- Interactive 3D skybox environments
- Personalized questionnaire flow to customize the tour
- Text-to-speech narration for accessibility

## Project Vision

"Vibe-coded" but deeply understod. 
The goal is to create an interface that feels responsive, alive, and premium. We are prioritizing visual excellence—modern typography, smooth gradients, and micro-animations.

## Local Development

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
